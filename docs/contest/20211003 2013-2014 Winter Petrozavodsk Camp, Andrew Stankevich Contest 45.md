## **B**

**solved by JLK**

### 题意

给定一个$[0,x]$上的分段线性函数，$\xi$是$[0,x]$上随机变量。

求$P(L \le \xi \le R|a \le f(\xi) \le b)\ge \alpha$的最小$R-L$的任意一对$L,R$​。​

$1 \le n \le 10^5$

### 题解

把函数上满足在$[a,b]$的区间找出来，显然答案必有一个方案，使$L,R$之一在找出来的区间端点上。尺取即可。

注意细节，函数可能是水平线。

$O(n)$
